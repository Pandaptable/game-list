Description


Momentum Mod is a first-person platforming game built on Valve’s Source Engine that centralizes movement-oriented game modes from various games like Counter-Strike, Team Fortress, Half-Life, and Titanfall. Speed is quite literally the name of the game, with each game mode offering a unique style of play centered around one shared goal: go fast!


  • Surf
  • Bunny Hop
  • Rocket Jump
  • Sticky Jump
  • Accelerated Hop
  • Tricksurf
  • Conc
  • Climb (KZ)
  • Parkour
  • Defrag


For seasoned veterans, Momentum aims to be as close to 1:1 as possible with the existing game modes found on servers, only with added convenience features and improvements as seen below:


  • Single-player client - no multiplayer game server connection required (no latency, ever)!
  • Client-side auto-bhop - no lag when jumping!
  • Surf ramp bug fixes - say goodbye to frustrating ramp bugs!
  • Slope jump fixes - consistent speed gained from jumping down slopes!
  • Original entities - easier mapping and interesting new opportunities for gamemodes!
  • Replays and analytics - for every run submitted!
  • Global leaderboards - for every map submitted!
  • Integrated map selector - easy browsing and one-click downloads for maps, all from in-game!
  • Hassle-free multiplayer - one-click multiplayer with Steam Lobbies (with community-focused dedicated servers coming later!)

For newer players, Momentum Mod aims to be the perfect entry point for those curious about the aforementioned game modes. Being a free-to-play platform means there’s absolutely no barrier to entry with learning any movement, so give it a try, and make sure to preserve your velocity!
